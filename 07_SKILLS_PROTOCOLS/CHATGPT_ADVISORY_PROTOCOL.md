# ChatGPT Advisory Protocol

## Purpose

Use ChatGPT as the advisory and planning layer of the project workflow.

## Use ChatGPT For

- problem framing
- clarification
- planning
- review
- instruction drafting
- risk identification

## Do Not Use ChatGPT Alone For

- repository file execution
- tracked branch changes
- authoritative file edits
- commit and push operations

## Expected Inputs

- project objective
- scope
- constraints
- current known facts
- questions that need clarification

## Expected Outputs

- clarified intent
- proposed approach
- explicit assumptions
- review findings
- draft instructions or issue text

## Advisory Rules

- Keep planning separate from execution.
- State uncertainty when repository facts are missing.
- Prefer concise instructions that Codex can execute directly.
- Route repository changes into GitHub Issues and Codex handoffs.
