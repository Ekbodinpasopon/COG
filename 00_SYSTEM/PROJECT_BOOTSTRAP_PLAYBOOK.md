# Project Bootstrap Playbook

## Objective

Create a new project from zero using this template without introducing private data.

## Required Inputs

- `[Project Name]`
- `[Repository Name]`
- `[Project Purpose]`
- `[Primary Domain]`
- `[Client Name]`
- `[Protected Systems]`
- `[Local Vault Path]`

## Bootstrap Sequence

1. Confirm project scope, privacy boundary, and publication intent.
2. Copy the template into `[Repository Name]`.
3. Update `TEMPLATE/CHATGPT_PROJECT_INSTRUCTIONS_TEMPLATE.md` for `[Project Name]`.
4. Create the Obsidian home note from `TEMPLATE/OBSIDIAN_LITE_HOME_TEMPLATE.md`.
5. Apply `TEMPLATE/PROJECT_FOLDER_STRUCTURE_TEMPLATE.md` to the new vault.
6. Create the first Codex handoff using `99_CODEX_HANDOFF/new-project-bootstrap-master-prompt.md`.
7. Run `TEMPLATE/SANITIZATION_CHECKLIST.md` before sharing the repository.

## Deliverables

- project instructions
- vault structure
- Codex handoff prompt
- bootstrap checklist status
- sanitization review status
