# BRAIN_INDEX

## Purpose

Map the project repository so contributors know where to place durable knowledge and workflow files.

## Recommended Sections

- Root files
- `00_SYSTEM/`
- `01_INBOX/`
- `02_IDEAS/`
- `03_MEMOS/`
- `04_LOGS/`
- `05_DECISIONS/`
- `06_TASKS/`
- `07_SKILLS_PROTOCOLS/`
- `99_ARCHIVE/`

## Mapping Format

- file or folder name
- purpose
- when to use it
