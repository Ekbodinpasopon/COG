# Example Project Scope

## Project Name

`[Example Project]`

## Objective

Create a Markdown-first operating structure for a new AI-assisted repository.

## In Scope

- repository root workflow files
- Obsidian-compatible folder map
- issue and handoff rules

## Out Of Scope

- application code
- packages
- automation scripts
- private operational data

## Acceptance Criteria

- required workflow files exist
- roles are clear
- template remains public-safe
