# Example Codex Handoff

## Repository

`[owner/example-repository]`

## Objective

Create the initial Markdown workflow files for a new AI-assisted repository.

## Scope

- root documentation
- workflow protocols
- template files

## Files to inspect

- `README.md`
- `AGENTS.md`

## Files to create/update

- `BRAIN_INDEX.md`
- `00_SYSTEM/PROJECT_BOOTSTRAP_PLAYBOOK.md`

## Constraints

- Markdown-only
- no code
- no private data

## Safety rules

- use placeholders only
- do not expand scope

## Acceptance criteria

- files are created
- routing rules are clear
- repository remains template-safe

## Report back format

1. Files created
2. Files modified
3. Assumptions used
4. Risks / rollback notes
5. Recommended next step
